# 🐰 兔子錢大冒險：我的理財日記

專為國小一至五年級設計的**課堂單機主控**互動桌遊網頁。老師單一操作、投影大螢幕，台下分組學生討論後由老師代為點擊。

## 功能
- 🎲 擲骰移動、環狀地圖（格數為 4 的倍數，剛好排滿）
- 五種屬性：💰金錢、💖幸福、🌱成長、💪健康（10 格顯示）、🧩任務碎片
- 健康或幸福歸零 → 強制休養（扣費、暫停一回合、恢復為 5）
- 格子事件：商店 / 挑戰 / 命運 / 愛心 / 銀行 / 公園 / 起點（經過起點發實體 200 元，系統不記錄）
- **後台可自由編輯**商店 / 挑戰 / 命運 / 愛心格的文字與數值
- 音效（Web Audio 合成）、全螢幕、防誤按重新整理
- 結算計分：`幸福×100 + 成長×200 + 集滿碎片+500 + 剩餘金錢`，頒獎 + 紙屑動畫

## 資料儲存
- 老師設定（角色 / 格子內容）自動存 **localStorage（本機）** ＋ **Firebase Firestore（雲端跨裝置同步）**。
- Firestore 文件：`config/teacherSettings`。

## 開發／部署
```bash
# 本機預覽
firebase serve

# 部署（Hosting + Firestore 規則）
firebase deploy
```

Firebase 專案：`rabbit-adventure-kai`
